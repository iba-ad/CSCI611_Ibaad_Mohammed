# Assignment 4

## Overview

This folder contains all deliverables for Assignment 4 of CSCI611.

## Contents

| File / Folder | Description |
|---------------|-------------|
| Source code (`.py` / `.ipynb`) | Implementation of the assignment |
| `report.pdf` | PDF report submitted to Canvas |
| `screenshots/` | Screenshots of successful execution (if not using a notebook) |
| `data/` / `images/` | Any image or dataset used for this assignment |

## Requirements

- Python 3.x
- Required libraries are listed at the top of the source file / notebook (install with `pip install <library>`)

## How to Run

### Python script

```bash
pip install -r requirements.txt   # if a requirements file is provided
python solution.py
```

### Jupyter Notebook

```bash
pip install jupyter
jupyter notebook assignment4.ipynb
```

Open the notebook in your browser, then select **Kernel → Restart & Run All** to execute all cells and reproduce the outputs, graphs, and tables.

> **Note:** Any images or datasets required by the code are stored in the `data/` folder (or alongside the notebook). Ensure the working directory is set to this folder before running.
